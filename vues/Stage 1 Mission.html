Analyse du besoin


Le site internet de la ville de Fontaine a été créé il y a déjà plusieurs avec Wordpress, le CMS (système de gestion de contenu).
Cependant, le développeur de celui-ci quitta la DSI de Fontaine, laissant derrière lui un site fonctionnel, mais sans aucune maintenance ni documentation le concernant.
Le site continue malgré tout d'être alimenté par l'équipe de communication de la ville.


Problématique


Créer un site internet à l'aide de WordPress semblable au site actuel de la ville.
Faire une documentation afin de premettre aux prochaines personnes de continuer mon travail.


Technologie utilisé


WordPress

Dashboard (Tableau de bord) : Permet d'avoir un aperçu du site

Posts (article) : gérer les différents article du site

Média : gérer tous les fichiers multimédias

Page : Similaire aux articles, mais permet de gérer des pages statiques

Commentaire : gérer les commentaires des utilisateurs

Apparence : permet de personnaliser l'apparence du site

Plugins (extensions) : gérer les extensions installé sur le site

Users (utilisateurs) : gérer les différents utilisateurs et leurs rôles
